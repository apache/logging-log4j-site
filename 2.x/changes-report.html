Release 2.15.0 – 2021-12-06

Type Changes By
Pattern layout no longer enables lookups within message text by default for cleaner API boundaries and reduced formatting overhead. The old 'log4j2.formatMsgNoLookups' which enabled this behavior has been removed as well as the 'nolookups' message pattern converter option. The old behavior can be enabled on a per-pattern basis using '%m{lookups}'. Fixes LOG4J2-3198. ckozak
Allow fractional attributes for size attribute of SizeBsaedTriggeringPolicy. Fixes LOG4J2-3194. Thanks to markuss. rgoers
